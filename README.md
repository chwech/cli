# 安装
```
npm i -g @chwech/cli
```

# 使用
```
// 列出模板列表
chwech tpl

// 初始化api-cloud-template
chwech init api-cloud

// caniuse查询
chwech caniuse es6

// 更新caniuse数据库
chwech caniuse update
```
